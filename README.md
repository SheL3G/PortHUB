
# PortHUB

PortHUB is a Python Port Scanner,PortHUB made for Educational Purposes and POC but feel 
free to use it on your machine / network.



## Usage

To start PortHUB open CMD / PowerShell / Terminal Window and the command above.
```python
python3 PortHUB.py
```
